Pick Serial Number Massive Update

Update the serial numbers of the given stock pick.



Use the standard import process of the model Lots/Serial Numbers and update the serial numbers in the given stock.move. A brief steps description:

Select the stock move to update.
Use the Lot/Serial Numbers Import Standard View

Headers Details **

  • product_ref: This is the Internal Reference defined in the product selected in the stock move line.
  • pick_number: The name of the pick to update.
  • name: The name of the new Lot/ Serial Number to link with the stock move line.
After import the new Lots/ Serial Numbers was created with the given names and the stock move lines are linked with. See the next picts.
